# 📊 Workforce Insights: HR Attrition Analysis Dashboard

## 1. Project Title / Headline
📊 **Workforce Insights: HR Attrition Analysis Dashboard**

An interactive analytics dashboard built using **Excel, MySQL, and Power BI** to explore employee attrition patterns and uncover workforce trends across departments, job roles, salary levels, and overtime.

---

# 2. Short Description / Purpose

The **HR Attrition Analysis Dashboard** is a data visualization project designed to analyze employee turnover and identify key factors influencing attrition.  
Using HR dataset insights, the dashboard enables users to examine workforce trends such as attrition rates, department distribution, salary impact, overtime patterns, and job role analysis to support data-driven HR decision making.

---

# 3. Tech Stack

| Tool | Purpose |
|-----|------|
| **Excel** | Data cleaning and preprocessing |
| **MySQL** | Data querying and analysis |
| **Power BI** | Dashboard creation and visualization |

---

# 4. Data Source

The dataset used for this project was sourced from **GitHub HR Analytics Dataset**.

It contains employee information such as:

- Employee demographics  
- Department and job role  
- Salary and compensation details  
- Overtime information  
- Attrition status  
- Years at company

This dataset enables the analysis of employee turnover trends and workforce insights.

---

# 5. Feature Highlights

### 📊 Attrition Overview
- Displays total employees, attrition count, and attrition rate.
- Provides a quick summary of workforce turnover.

### 🏢 Department Analysis
- Identifies departments with the highest attrition.
- Helps HR teams focus on departments with retention issues.

### 💼 Job Role Insights
- Analyzes attrition across different job roles.
- Highlights roles with higher employee turnover.

### 💰 Salary Impact Analysis
- Examines how salary levels influence employee attrition.
- Shows distribution of employees across salary bands.

### ⏱️ Overtime Analysis
- Compares attrition between employees who work overtime and those who do not.
- Helps identify workload-related attrition factors.

### 👥 Employee Demographics
- Provides insights into attrition by age groups and tenure.
- Helps understand workforce composition and retention trends.

# 8. Business Insights

Some key insights derived from the dashboard:

- Employees working **overtime show higher attrition rates**.
- Certain **departments and job roles experience higher turnover**.
- **Lower salary bands** tend to have higher attrition levels.
- Employees with **fewer years at the company are more likely to leave**.

These insights can help organizations improve **employee retention strategies and workforce planning**.
